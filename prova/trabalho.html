  1. background-color: Define um cor de fundo de um elemento.
  2. font-family: Define a família de fonte a ser usada para o texto.
  3. max-width: Define a largura máxima de um elemento.
  4. margin: Defina como margens de um elemento.
  5. padding: Defina o preenchimento interno de um elemento.
  6. border-radius: Define o raio do arredondamento das bordas de um elemento.

  8. text-align: Alinha o texto dentro de um elemento.
  9. display: Defina o comportamento de exibição de um elemento.
  10. width: Define a largura de um elemento.
  11. margin-bottom: Define a margem inferior de um elemento.
  12. border: Defina as propriedades da borda de um elemento.
  13. cursor: Define o cursor do mouse quando passa sobre um elemento.
  14. :hover: Seletor que aplica estilos quando o elemento está no estado de "hover" (quando o mouse está sobre ele).
